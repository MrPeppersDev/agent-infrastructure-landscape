Luminance Institutional Memory

When to use

Optimised for: matter-scoped privilege + audit + firm-precedent memory

Anti-fit: not for non-legal verticals; not for self-represented litigants

Pros & cons

Pros

Self-supervised model trained on a firm's prior contracts — institutional memory captures firm-specific patterns competitors don't see.

Cons

Institutional memory is a property of long Luminance deployments; benefit accrues over time, not on day one.
