NIAH (Needle in a Haystack)

https://github.com/gkamradt/LLMTest_NeedleInAHaystack

Places a known sentence ("needle") at varying positions inside a long irrelevant document ("haystack") and prompts the model to retrieve it. Tests context lengths from 2K to model max. Greg Kamradt, 2023, no formal venue.

Pros & cons

Pros

Original needle-in-a-haystack benchmark — defined the long-context recall test paradigm.

Cons

Single-pattern test; commercial models now ace it; less informative for modern memory architectures.

Claims & capabilities

De facto first-pass filter for long-context LLM releases. Now considered necessary-but-insufficient — most frontier models score near-perfect.
