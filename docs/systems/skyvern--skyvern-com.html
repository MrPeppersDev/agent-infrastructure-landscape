Skyvern

https://www.skyvern.com/

Open-source autonomous browser agent that uses computer vision + LLMs to navigate websites. Workflow-based: agents follow recorded plans that adapt to layout changes. Skyvern Inc. raised $4.5M seed (Sep-2024, Felicis lead). Self-hostable; cloud version on skyvern.com.
