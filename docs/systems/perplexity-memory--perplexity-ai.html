Perplexity Memory

https://www.perplexity.ai/help-center/en/articles/10968016-memory

Auto-infers facts from conversations and surfaces them in future responses (Pro subscribers). Cross-model: memories persist when user switches underlying model mid-session. Enterprise Memory adds org-scoped retrieval.

Pros & cons

Pros

Memory layered onto a search-grounded answer engine means recalled facts arrive with citations by construction.

Cons

Primarily helps repeat-search scenarios; less useful as general assistive memory than competitors.

Claims & capabilities

95% recall (Feb 2026 update, up from 77%) while reducing memory count by ~50%.
