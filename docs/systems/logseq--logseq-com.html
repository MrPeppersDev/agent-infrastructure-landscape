Logseq

https://logseq.com/

Open-source local-first outliner. Notes stored as Markdown / Org-mode on disk. AI integration via community plugins (e.g. logseq-ai for chatting with the graph). Graph view + backlinks; daily-journal-driven workflow.

Pros & cons

Pros

Open-source PKM with block-graph model; memory benefits from explicit knowledge-graph structure.

Cons

Less mature AI integrations than Notion / Obsidian; community-driven so velocity varies.

Claims & capabilities

Open-source local-first PKM in active community; common comparison anchor against Tana / Obsidian / Roam in 2025 PKM space; daily-notes + outliner block model; users often pair with Tana as complement vs primary tool
