smol-developer

https://github.com/smol-ai/developer

Early (2023) OSS coding agent by Swyx — minimalist single-prompt-to-codebase loop. ~12k★. More artifact-than-product now (subsumed by Cline / Aider / Claude Code) but historically significant — defined the "smol" school of single-file coding agents.
