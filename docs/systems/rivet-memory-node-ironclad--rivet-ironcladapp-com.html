Rivet Memory Node (Ironclad)

https://rivet.ironcladapp.com/

Open-source visual AI programming environment. Memory is a first-class graph node: stores text block, supports stable IDs (saving same ID overwrites enabling updates), injects stored text into downstream nodes when relevant. Memory is designed and debugged visually.
