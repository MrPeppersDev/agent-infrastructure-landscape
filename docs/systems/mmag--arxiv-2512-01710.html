Claims & capabilities

Mixed Memory-Augmented Generation framework organizing LLM-agent memory into five interacting layers (conversational, long-term user, episodic and event-linked, sensory and context-aware, short-term working memory); implementation in Heero conversational agent shows encrypted long-term bios and conversational history improve engagement and retention; abstract provides no quantitative metrics
