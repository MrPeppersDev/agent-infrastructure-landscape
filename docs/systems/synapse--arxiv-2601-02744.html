SYNAPSE

https://arxiv.org/abs/2601.02744

Memory is a dynamic graph; retrieval fires spreading activation across edges rather than vector similarity. Lateral inhibition suppresses competing activations; temporal decay modulates edge weights. Episodic and semantic stores maintained separately.
