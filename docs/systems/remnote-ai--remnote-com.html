When to use

Optimised for: personal-knowledge-management UX + AI-augmented note-taking

Anti-fit: not for enterprise / programmatic use; not for compliance-regulated workloads

Pros & cons

Pros

Built on a spaced-repetition substrate — memory has explicit retention scheduling, unlike most note tools that retrieve on demand.

Cons

Smaller user base than Notion / Obsidian; AI features lag general-purpose PKM tools.

Claims & capabilities

Free; Pro $8/mo; Pro+AI ~$18/mo (20k AI credits).
