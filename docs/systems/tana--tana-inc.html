Tana

https://tana.inc/

Outliner + structured-data hybrid (Roam/Logseq + Notion-database influences). "Supertags" turn freeform nodes into queryable databases; Tana AI invokes on any node to summarise, generate action items, or extract structured fields (e.g. Meeting node with attendees + decisions + follow-ups).

When to use

Optimised for: personal-knowledge-management UX + AI-augmented note-taking

Anti-fit: not for enterprise / programmatic use; not for compliance-regulated workloads

Pros & cons

Pros

Supertag + node graph model gives Tana a structurally richer memory substrate than flat-file notes.

Cons

Steep learning curve; smaller user base than Notion / Obsidian; closed system.

Claims & capabilities

Released end-September 2022 (vs Logseq/Obsidian who preceded by 2+ years); rapid PKM-community adoption with vocal advocate base + steady migration from Notion / Roam / Logseq / Obsidian; supernode-based outliner with first-class AI integration; primarily desktop / web
