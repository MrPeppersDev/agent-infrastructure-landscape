OpenAI Realtime API

https://platform.openai.com/docs/guides/realtime

OpenAI's Realtime API for native speech-to-speech multimodal interaction with GPT-4o — launched Oct 2024 (DevDay). Skips the STT → LLM → TTS pipeline by passing speech directly to GPT-4o. The platform substrate that Vapi/Retell/Bland integrate as one of their LLM choices.

When to use

Optimised for: Native speech-to-speech with GPT-4o; lowest-latency LLM voice tier

Anti-fit: Stateless platform — caller builds orchestration, memory, telephony adapter

Pros & cons

Pros

Native speech-to-speech (skips STT/TTS); OpenAI brand + price/perf; WebRTC support.

Cons

Audio token pricing is expensive at scale; stateless (caller builds memory/orchestration); OpenAI-only model choice.

Claims & capabilities

Launched DevDay Oct-2024; GPT-4o speech-to-speech native; WebSocket + WebRTC support
