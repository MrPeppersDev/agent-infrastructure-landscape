Claims & capabilities

Modular multi-agent memory system with six memory types (Core, Episodic, Semantic, Procedural, Resource, Knowledge Vault); 35% higher accuracy than RAG baseline with 99.9% storage reduction on ScreenshotVQA; state-of-the-art 85.4% on LOCOMO; supports multimodal screenshots (~20,000 per sequence)
