Claims & capabilities

Frozen backbone LLM + adaptive residual side-network as memory retriever/reader; ~65K token memory bank. Headline: 40.5% identification accuracy on ChapterBreak vs. GPT-3 28% with 313x fewer parameters; -1.38 to -1.62 perplexity on PG-22; baselines: Memorizing Transformer (MemTRM) and reproduced GPT-2 (407M Alibi); primary datasets: ChapterBreak, PG-22, ArXiv papers, plus ICL on SST-2/MR/Subj/SST-5/MPQA.
