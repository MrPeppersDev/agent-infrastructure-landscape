Obsidian Smart Connections

https://github.com/brianpetro/obsidian-smart-connections

Obsidian community plugin. Generates embeddings locally with a bundled model (TaylorAI/bge-micro-v2, 384-dim); chat with notes + see semantically related links. No API key required for local; supports 100+ cloud models optionally. Privacy-first.

Pros & cons

Pros

Plugin to the most-extensible PKM tool — memory layered on top of your local markdown vault means full data ownership.

Cons

Obsidian-tied; community plugin so quality depends on maintainer; setup more involved than first-party AI.

Claims & capabilities
