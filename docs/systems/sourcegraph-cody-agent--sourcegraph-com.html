Sourcegraph Cody (agent)

https://sourcegraph.com/cody

Sourcegraph's coding assistant — IDE plugin (VS Code, JetBrains) plus web. Distinguishes itself with Sourcegraph's code-search index as the context provider (cross-repo, codebase-wide retrieval). Agentic chat added 2024; runs Claude / GPT / Gemini.
