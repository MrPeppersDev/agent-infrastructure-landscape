Claims & capabilities

Tool knowledge integrated directly into LLM parameters by representing each tool as a unique token; tool calls and arguments generated as part of standard token prediction without retrieval steps; handles vast tool inventories without context length constraints; superior performance in tool retrieval and autonomous task completion; tested with over 47,000 tools; ICLR 2025
