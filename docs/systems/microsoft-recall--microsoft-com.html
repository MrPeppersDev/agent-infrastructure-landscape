Microsoft Recall

https://www.microsoft.com/en-us/windows/copilot-plus-pcs

Windows-native screen-recording memory. Continuously indexes activity across apps, documents, browsing; timeline UI for revisiting work. Distinct from Copilot Memory.

When to use

Anti-fit: not for enterprise / programmatic use; not for compliance-regulated workloads

Pros & cons

Pros

Most aggressive ambient capture of any major OS — every screen seen is searchable, then queryable.

Cons

Drew significant privacy backlash on launch (May 2024); only Copilot+ PC hardware supported.

Claims & capabilities

Originally announced May 2024, retracted within weeks following security/privacy backlash. Re-released April 2025 to Windows Insiders Release Preview Channel with stronger encryption, biometric Windows Hello auth, opt-in default, automated content filtering of credit-card / banking / password fields. Brave, Signal, AdGuard added explicit Recall-blocking. Available on Copilot+ PCs only
