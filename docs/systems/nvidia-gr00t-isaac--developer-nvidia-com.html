NVIDIA GR00T / Isaac

https://developer.nvidia.com/isaac/gr00t

NVIDIA Project GR00T (Mar-2024) — general-purpose humanoid foundation model + Isaac simulation stack + Jetson Thor compute. GR00T N1 model released open March 2025. NVIDIA's full-stack play: model + sim + compute + training cloud (Cosmos). Major partners include Figure, Apptronik, Boston Dynamics, Agility.
