Pros & cons

Apple Notes + Apple Intelligence

Pros: On-device intelligence keeps personal-note context private; integrates across Apple's notes, mail, calendar memory.

Cons: Apple-only; AI feature set lags GPT-class competitors and is conservative about cross-app inference.

RemNote AI

Pros: Built on a spaced-repetition substrate — memory has explicit retention scheduling, unlike most note tools that retrieve on demand.

Cons: Smaller user base than Notion / Obsidian; AI features lag general-purpose PKM tools.
